# KNOWLEDGE — Wiedza domenowa AI + SAP

## AI Agent Frameworks

| Framework | Opis | Zastosowanie SAP |
|-----------|------|-----------------|
| **LangGraph** | Graph-based agent orchestration (LangChain) | Workflowy BPMN, multi-step processing |
| **CrewAI** | Role-based multi-agent collaboration | Zespoły agentów per moduł SAP |
| **AutoGen** | Microsoft multi-agent conversation | Auto-rozwiązywanie problemów integracyjnych |
| **OpenAI Agents SDK** | Oficjalny SDK OpenAI do agentów | Proste taski, integracja z GPT |
| **MCP (Model Context Protocol)** | Standard Anthropic do tool-server | Uniwersalny most między LLM a SAP API |
| **smolagents** | HuggingFace lekkie agenty | Eksperymenty, małe taski |

## SAP AI Stack

- **SAP AI Core** — zarządzanie modelami ML w BTP
- **Generative AI Hub** — hosting LLM (GPT, Llama, Claude, Gemini)
- **Joule** — wbudowany asystent AI SAP
- **SAP BTP Orchestration** — prompt templates + grounding + tools
- **Document Information Extraction** — OCR + strukturyzacja
- **AI Business Services** — rozpoznawanie encji, tłumaczenia
